# Hebrew Technical Vocabulary for Tech Interviews

Common Hebrew technical terms used in Israeli tech interviews, organized by category. While most Israeli tech interviews use a mix of Hebrew and English, familiarity with these terms shows cultural comfort and helps in day-to-day team communication.

**Note**: Many English technical terms are used as-is in Hebrew (transliterated), especially for newer concepts. This guide includes both native Hebrew terms and common transliterations.

---

## 7. Israeli Tech Slang and Culture

| Term | Meaning | Usage |
|------|---------|-------|
| לעשות דיפלוי (La'asot deploy) | To deploy | "Let's do a deploy" |
| לפקס (Lefaks) | To fix | Very common verb form |
| לדבג (Ledabeg) | To debug | Hebraicized "debug" |
| למרג'ג' (Lemargej) | To merge | "Merge the branch" |
| הרמת דגל (Haramat Degel) | Raising a flag / flagging an issue | "I want to raise a flag about..." |
| לעלות לפרודקשן (La'alot LeProduction) | To deploy to production | "When are we going to production?" |
| לשרוף ספרינט (Lisrof Sprint) | To burn through a sprint | Finishing sprint tasks quickly |
| דיזיין דוק (Design Doc) | Design document | "Write a design doc first" |
| הפיצ'ר בוער (HaFeature Bo'er) | The feature is urgent | "This feature is on fire" |
| POC (פי-או-סי) | Proof of concept | "Let's do a quick POC" |
| MVP (אמ-וי-פי) | Minimum viable product | Very common in startups |
| לעוף (La'uf) | To be fired / to leave quickly | "He flew (was fired)" |
| גיקי (Giki) | Geeky / technical | Positive connotation in tech |
| חברה קטנה (Chevra Ktana) | Small startup | Informal description |
| האקתון (Hackathon) | Hackathon | Common at Israeli companies |
| לסנכרן (Lesanchren) | To sync | "Let's sync on this" |
| אונבורדינג (Onboarding) | Onboarding | Used as-is |
| פידבק (Feedback) | Feedback | Used as-is, very direct in Israeli culture |

## 8. Common Interview Phrases

| Hebrew | English | When Used |
|--------|---------|-----------|
| ספר/י לי על עצמך | Tell me about yourself | Opening question |
| מה הניסיון שלך עם...? | What's your experience with...? | Technical screening |
| איך היית ניגש/ת לבעיה הזו? | How would you approach this problem? | System design |
| מה הסיבוכיות? | What's the complexity? | After coding solution |
| אפשר לשפר את זה? | Can you improve this? | Follow-up to coding |
| תסביר/י את התהליך | Explain the process | Architecture discussion |
| מה הפשרות? | What are the trade-offs? | System design |
| למה בחרת את הגישה הזו? | Why did you choose this approach? | Design justification |
| יש שאלות? | Any questions? | End of interview |
| מה חשוב לך בתפקיד? | What's important to you in a role? | Behavioral |
| תן/י דוגמה ל... | Give an example of... | Behavioral |
| מה הציפיות שלך לשכר? | What are your salary expectations? | HR screen |
