# Hebrew Document Glossary for Genealogical Research

## Birth Certificate (תעודת לידה)

| Hebrew | Transliteration | English |
|--------|----------------|---------|
| שם פרטי | Shem prati | First name |
| שם משפחה | Shem mishpacha | Family name |
| תאריך לידה | Ta'arikh leida | Date of birth |
| מקום לידה | Makom leida | Place of birth |
| שם האב | Shem ha'av | Father's name |
| שם האם | Shem ha'em | Mother's name |
| מספר זהות | Mispar zehut | ID number |
| מין | Min | Gender (זכר = male, נקבה = female) |
| דת | Dat | Religion |
| לאום | Le'om | Nationality/Ethnicity |
| ארץ לידה | Eretz leida | Country of birth |

## Marriage Certificate (תעודת נישואין)

| Hebrew | Transliteration | English |
|--------|----------------|---------|
| חתן | Chatan | Groom |
| כלה | Kala | Bride |
| תאריך הנישואין | Ta'arikh hanisu'in | Marriage date |
| מקום הנישואין | Makom hanisu'in | Place of marriage |
| מסדר הקידושין | Mesader hakidushin | Officiating rabbi |
| עדים | Edim | Witnesses |
| כתובה | Ketuba | Marriage contract |
| שם האב של החתן | Shem ha'av shel hachatan | Groom's father's name |
| שם האב של הכלה | Shem ha'av shel hakala | Bride's father's name |
| גירושין | Gerushin | Divorce |
| גט | Get | Divorce document |

## Immigration Documents

| Hebrew | Transliteration | English |
|--------|----------------|---------|
| עולה | Oleh (m) / Olah (f) | Immigrant (to Israel) |
| עלייה | Aliyah | Immigration to Israel |
| תעודת עולה | Te'udat oleh | Immigrant certificate |
| ארץ מוצא | Eretz motza | Country of origin |
| תאריך עלייה | Ta'arikh aliyah | Date of immigration |
| נמל הגעה | Namal haga'a | Port of arrival |
| שם האנייה | Shem ha'aniya | Ship name |
| מעברה | Ma'abara | Transit camp |
| סל קליטה | Sal klita | Absorption basket (financial aid) |
