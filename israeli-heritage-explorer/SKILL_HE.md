# חוקר מורשת ישראלית

## בעיה

מחקר של היסטוריה ותרבות ישראלית-יהודית דורש ניווט במיליוני רשומות בארכיון הדיגיטלי של הספרייה הלאומית. בלי גישה מובנית, חוקרים מבזבזים שעות בגלישה ידנית באתר הספרייה, מפספסים מקורות ראשוניים חשובים, ולא מצליחים לחבר בין חומרים קשורים מאוספים שונים. הסקיל הזה נותן גישה ישירה ל-API של הספרייה הלאומית ומייצר דוחות מחקר מסודרים עם ציטוטים מדויקים.

## הוראות

### סקירה

הסקיל מחפש בארכיון הדיגיטלי של הספרייה הלאומית דרך ה-OpenLibrary API. הספרייה הלאומית מחזיקה מיליוני פריטים: כתבי יד עתיקים, מסמכים ממשלתיים, תצלומים היסטוריים, מפות, הקלטות מוזיקליות, עדויות בעל-פה ופרסומים מדעיים.

### מתי להשתמש

- כשמשתמש שואל על היסטוריה ישראלית (כל תקופה: עות'מאנית, מנדט בריטי, ראשית המדינה, מודרנית)
- כשרוצים מקורות ראשוניים על מורשת יהודית
- כשמחפשים תצלומים או מפות היסטוריים של מקומות בישראל
- כשחוקרים גלי עלייה לישראל
- כשרוצים לחקור היסטוריה של קהילה, שכונה או מוסד ספציפי
- כשצריך ביבליוגרפיה מוערת על נושא תרבותי ישראלי

### גישה ל-API

ה-API של הספרייה הלאומית הוא חינמי. כתובת הבסיס:

```
https://api.nli.org.il/openlibrary/search
```

**אימות:** נדרש מפתח API שמועבר כפרמטר `api_key`. אפשר לקבל מפתח חינמי ב-https://api2.nli.org.il/signup/. שרת ה-MCP של `nli-search` מגיע עם מפתח מבקרים ברירת מחדל.

**שרת MCP מומלץ:** התקינו את `nli-search` לגישה ישירה ל-API עם שאילתות בשפה טבעית, שליפת תמונות IIIF וסטרימינג מדיה.

| שרת MCP | מה הוא מספק |
|---------|------------|
| [nli-search](https://agentskills.co.il/he/mcps/nli-search) | חיפוש בשפה טבעית, שליפת תמונות IIIF, סטרימינג מדיה, עיבוד תוצאות בקבוצות |

### פרמטרים לחיפוש

ה-API מקבל שאילתות מובנות בפורמט `field,operator,value`:

| פרמטר | תיאור | דוגמה |
|--------|-------|-------|
| `query` | חיפוש ראשי (field,operator,value) | `subject,contains,Tel Aviv history` |
| `material_type` | סינון לפי סוג | `books`, `images`, `manuscripts`, `maps`, `audio`, `videos`, `articles`, `journals`, `rareBooks` |
| `language` | סינון לפי שפה | `heb`, `eng`, `ara`, `yid`, `lad` |
| `creator` | סינון לפי יוצר | `creator,contains,Ben Gurion` |
| `subject` | סינון לפי נושא | `subject,contains,Zionism` |
| `publication_year_from` / `publication_year_to` | טווח תאריכים | `1920` עד `1948` |
| `rows` | תוצאות לעמוד (מקס' 500) | `50` |
| `output_format` | פורמט תשובה | `json` או `xml` |

### תהליך מחקר

**שלב 1: הגדרת היקף המחקר**

בררו או הסיקו:
- תקופה היסטורית (עות'מאנית 1517-1917, מנדט בריטי 1917-1948, ראשית המדינה 1948-1967, מודרנית 1967-היום)
- מיקוד גאוגרפי (עיר ספציפית, אזור, או כל ישראל)
- סוגי חומרים (תצלומים, מסמכים, ספרים, מפות, אודיו)
- שפה מועדפת (עברית, אנגלית, ערבית, יידיש, לאדינו)

**שלב 2: חיפוש ב-API**

בנו שאילתות ממוקדות. לתוצאות הטובות ביותר, שלבו חיפוש נושא עם סינון סוג חומר ותאריכים:

```
# תצלומים היסטוריים של יפו בתקופת המנדט
query=subject,contains,Jaffa&material_type=images&publication_year_from=1917&publication_year_to=1948

# כתבי יד בעברית על ירושלים
query=subject,contains,Jerusalem&material_type=manuscripts&language=heb

# כתבי דוד בן-גוריון
query=creator,contains,Ben Gurion&material_type=books&language=heb
```

**שלב 3: עיבוד תוצאות**

כל תוצאה מכילה:
- `title` -- שם הפריט (לרוב בעברית)
- `type` -- סוג חומר (ספר, ארכיון, תמונה וכו')
- `date` -- תאריך יצירה/פרסום
- `publisher` -- גוף מפרסם
- `language` -- שפת התוכן
- `recordid` -- מזהה ייחודי בספרייה הלאומית
- `@id` -- קישור ישיר לפריט באתר nli.org.il
- `linkToMarc` -- קישור IIIF לתמונות

**שלב 4: יצירת תוצר מחקרי**

עצבו את הממצאים לאחד מהפורמטים הבאים:

**ביבליוגרפיה מוערת:**
```markdown
## ביבליוגרפיה מוערת: [נושא]

### מקורות ראשוניים

1. **[כותרת]** ([תאריך])
   - סוג: [כתב יד/תצלום/מפה/וכו']
   - שפה: [עברית/אנגלית/וכו']
   - קישור לספרייה הלאומית: [URL ישיר]
   - רלוונטיות: [משפט-שניים המסביר את חשיבות המקור]
```

**דוח מחקרי:**
```markdown
## דוח מחקרי: [נושא]

### הקשר היסטורי
[2-3 פסקאות המספקות הקשר על בסיס המקורות]

### מקורות ראשוניים מרכזיים
[רשימה ממוספרת עם הערות]

### חומרים חזותיים
[תצלומים, מפות עם תיאורים וקישורים]

### ציר זמן
[רשימה כרונולוגית של אירועים מרכזיים עם ציטוטי מקור]
```

### אוספים מרכזיים בספרייה הלאומית

| אוסף | תוכן | מונחי חיפוש |
|-------|-------|------------|
| אוסף מפות ערן לאור | 30,000+ מפות היסטוריות של ארץ ישראל | `material_type=maps` + מונחים גאוגרפיים |
| אוסף דיוקנאות שוודרון | דיוקנאות היסטוריים של אישים יהודיים | `subject,contains,Schwadron` + `material_type=images` |
| הארכיון הלאומי לקול | מוזיקה, עדויות בעל-פה, שידורי רדיו | `material_type=audio` |
| מכון כתבי היד | כתבי יד בעברית, ערבית ושפות נוספות | `material_type=manuscripts` |
| אוסף שאפל | מסמכים מתקופת השלטון העות'מאני והמנדט הבריטי | `subject,contains,Palestine` + טווח תאריכים |
| אוספי עיתונות | עיתונים היסטוריים בעברית | `material_type=journals` |

### גישה לתמונות IIIF

לחומרים חזותיים, הספרייה הלאומית מספקת נקודות קצה IIIF:

- **Image API:** `https://iiif.nli.org.il/IIIFv21/{identifier}/full/max/0/default.jpg`
- **Manifest API:** `https://iiif.nli.org.il/IIIFv21/MARC/{recordid}/manifest`

השתמשו ב-manifest כדי לקבל את כל התמונות הזמינות לפריט מרובה דפים (כתב יד, ספר, עיתון).

### טיפים למחקר לפי תקופה

**התקופה העות'מאנית (1517-1917):**
- חפשו בטורקית עות'מאנית, ערבית ועברית
- נושאים מרכזיים: בעלות על קרקע (טאבו), בתי דין שרעי, קהילות יהודיות (היישוב)

**המנדט הבריטי (1917-1948):**
- עשיר במסמכים מנהליים באנגלית
- חפשו `subject,contains,Palestine` לחומרי המנדט
- נושאים: תעודות עלייה, רכישות קרקע, תכנון עירוני, עיתונות

**ראשית המדינה (1948-1967):**
- פרסומים ממשלתיים, רשומות הכנסת, מסמכים צבאיים
- תיעוד גלי העלייה הגדולים (עלייה המונית)
- תכנון עיירות פיתוח, הקמת מוסדות תרבות

**התקופה המודרנית (1967-היום):**
- הקלטות עדויות בעל-פה, צילום עכשווי
- מחקר אקדמי, תיעוד תרבותי

## מלכודות נפוצות

1. **אורך שאילתה מינימלי:** ה-API דורש מונחי חיפוש ארוכים מ-2 תווים. אותיות בודדות או מילים קצרות מאוד מחזירות שגיאה. השתמשו במונחי נושא מתארים.

2. **חיפוש בעברית מול אנגלית:** הרבה רשומות בספרייה הלאומית מכילות מטאדאטה בעברית בלבד. אם חיפוש באנגלית מחזיר מעט תוצאות, נסו שוב בעברית. ה-API לא מתרגם אוטומטית בין שפות.

3. **פורמט תאריכים לא אחיד:** שדה `date` בתוצאות משתמש בפורמטים שונים: חלק מהפריטים בפורמט `YYYYMMDD`, אחרים בפורמט `YYYY`, וחלק בטקסט חופשי כמו `[circa 1930]` או `תרצ"ב`. נתחו תאריכים בזהירות.

4. **סינון סוג חומר:** הפרמטר `material_type` מחמיר. שימוש ב-`photos` במקום `images` מחזיר אפס תוצאות. תמיד השתמשו בערכים המדויקים: `books`, `images`, `manuscripts`, `maps`, `audio`, `videos`, `articles`, `journals`, `rareBooks`.

5. **הגנת Cloudflare:** ה-API מוגן על ידי Cloudflare. בקשות מכתובות IP של שרתי ענן עלולות להיחסם. ה-API עובד באופן אמין מכתובות IP ביתיות וסביבות פיתוח מקומיות.

## פתרון בעיות

### שגיאת "Please use more than two characters"
השאילתה קצרה מדי. השתמשו במונח ארוך ומתאר יותר.

### תוצאות ריקות לנושא ידוע
נסו אסטרטגיות חיפוש שונות:
1. החליפו בין מונחי חיפוש באנגלית ובעברית
2. הרחיבו את טווח התאריכים
3. הסירו את סינון `material_type` כדי לחפש בכל הסוגים
4. השתמשו ב-`subject,contains` במקום שאילתה חופשית

### תשובת 403 Forbidden
מפתח ה-API לא תקין או פג תוקף. קבלו מפתח חינמי חדש ב-https://api2.nli.org.il/signup/.
